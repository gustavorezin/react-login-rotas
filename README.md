## 📝 Começando

1. **Instale as dependências**: `npm install` ou `yarn`
2. **Variáveis de ambiente**: Copie `.env.example` para um novo `.env.local`
3. **Comece a desenvolver**: `npm run start:json-server (api) & npm run dev (frontend)`
